# 国风消消乐 - 产品需求文档

## 1. 项目概述

| 项目 | 内容 |
|-----|------|
| 名称 | 国风消消乐小游戏 |
| 类型 | 休闲益智类网页游戏 |
| 技术 | HTML5 + CSS3 + JavaScript |
| 部署 | 纯前端静态部署（本地/云端） |

**一句话定位**：一款融合中国传统美学元素的经典三消休闲游戏。

## 2. 核心玩法

- **棋盘**：8×8
- **规则**：3个及以上相同元素连成一线即可消除
- **操作**：点击选择 + 交换相邻元素
- **关卡目标**：达成目标分数通关

**元素（6种）**：

| 元素 | 符号 | 颜色 |
|-----|------|------|
| 灯笼 | 🏮 | 朱红 |
| 玉佩 | 🀄 | 翠绿 |
| 瓷器 | 🏺 | 青蓝 |
| 铜钱 | 💰 | 金黄 |
| 祥云 | ☁️ | 奶白 |
| 梅花 | 🌸 | 粉红 |

## 3. 功能模块

### 3.1 核心功能
- ✅ 三消消除逻辑
- ✅ 连锁检测与连击加成
- ✅ 关卡系统（10关，难度递增）
- ✅ 步数限制
- ✅ 分数统计
- ✅ 胜利/失败判定

### 3.2 社交功能
- ✅ 本地排行榜（Top 10）
- ⏳ 云端排行榜（可选）
- ⏳ 多人在线（最多30人，可选）

### 3.3 系统功能
- ✅ 本地存储（localStorage）
- ✅ 音效开关
- ✅ 昵称设置
- ⏳ 体力系统（可选）
- ⏳ 道具商城（可选）

## 5. 数据结构

```javascript
// localStorage 键名
'chineseMatch3_settings'   // 用户设置
'chineseMatch3_leaderboard' // 排行榜
'chineseMatch3_maxLevel'  // 最高关卡

// 游戏状态
{
  board: [],        // 8x8棋盘
  score: 0,         // 当前分数
  moves: 25,        // 剩余步数
  level: 1,         // 当前关卡
  targetScore: 1000 // 目标分数
}

// 关卡配置
[
  { target: 1000, moves: 25 },
  { target: 1500, moves: 25 },
  { target: 2000, moves: 24 },
  ...
]
```
