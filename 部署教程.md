# 国风消消乐 - 完整部署教程

---

## 一、本地运行（无需服务器）

### 方式1：直接打开
```
找到 index.html 文件，双击在浏览器中打开
```

### 方式2：Python HTTP 服务器
```bash
# 进入游戏目录
cd /path/to/游戏目录

# 启动服务器
python -m http.server 8080

# 浏览器访问 http://localhost:8080
```

### 方式3：Node.js
```bash
# 安装 serve
npm install -g serve

# 启动
serve .

# 浏览器访问显示的地址
```

## 二、云端部署（免费）

### 方案1：Vercel（推荐）

1. **准备文件结构**
```
国风消消乐/
├── index.html    # 游戏主文件
└── vercel.json  # 配置文件
```

2. **创建 vercel.json**
```json
{
  "rewrites": [
    { "source": "/(.*)", "destination": "/index.html" }
  ]
}
```

3. **部署**
```bash
# 安装 Vercel CLI
npm i -g vercel

# 登录
vercel login

# 部署
vercel

# 或者拖拽文件夹到 https://vercel.com/drop
```

### 方案2：Netlify

1. 访问 https://netlify.com/drop
2. 拖拽整个游戏文件夹到网页
3. 自动部署完成，获得网址

### 方案3：GitHub Pages

1. 创建 GitHub 仓库
2. 上传 index.html
3. Settings → Pages → 选择 main branch
4. 获得网址：`https://你的用户名.github.io/仓库名/`

## 三、云端服务器部署（多人在线 + 排行榜）

### 环境要求
- Node.js 16+
- npm 或 yarn

### 步骤1：安装依赖

```bash
cd server
npm install
```

### 步骤2：启动服务器

```bash
# 开发模式（本地测试）
npm start

# 服务器运行在 http://localhost:3000
```

### 步骤3：配置前端连接

编辑 `index.html` 中的服务器地址：

```javascript
// 找到这行，替换为你的服务器地址
const CLOUD_API = 'https://your-server.com/api';
const wsUrl = 'wss://your-server.com';
```

### 步骤4：云端服务器部署

#### 方案A：Railway（免费）
1. 注册 https://railway.app
2. New → GitHub Repo → 选择项目
3. 自动部署

#### 方案B：Render
1. 注册 https://render.com
2. New → Web Service
3. 连接到 GitHub 仓库
4. 启动命令：`node server/server.js`

#### 方案C：阿里云/腾讯云服务器
```bash
# 1. 上传 server 文件夹到服务器
# 2. SSH 登录服务器
ssh root@你的服务器IP

# 3. 安装 Node.js
curl -fsSL https://deb.nodesource.com/setup_18.x | sudo -E bash -
sudo apt-get install -y nodejs

# 4. 上传文件并启动
cd /path/to/server
npm install
pm2 start server.js

# 5. 配置 Nginx 反向代理
```

## 四、30人并发配置

### 性能要求

| 人数 | CPU | 内存 | 带宽 |
|-----|-----|------|-----|
| 1-30人 | 1核 | 512MB | 1Mbps |

### Nginx 配置（生产环境）

```nginx
upstream game_server {
    server localhost:3000;
}

server {
    listen 80;
    server_name your-domain.com;

    location / {
        root /path/to/frontend;
        index index.html;
    }

    location /api {
        proxy_pass http://game_server;
    }

    location /socket.io {
        proxy_pass http://game_server;
        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection "upgrade";
        proxy_set_header Host $host;
    }
}
```

## 六、常见问题

### Q1: 多人游戏无法连接？
- 检查服务器是否正常运行
- 检查防火墙是否开放端口 3000
- 检查前端配置的服务器地址是否正确

### Q2: 云端排行榜显示不了？
- 确认服务器 API 是否可访问
- 检查浏览器控制台错误信息

### Q3: 加载很慢？
- 使用 CDN 加速静态资源
- 启用 Gzip 压缩
